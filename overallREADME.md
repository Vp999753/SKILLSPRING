**SKILLSPRING** is a collection of hands-on Java and Spring Framework exercises completed as part of Enterprise Java Development training.

The repository demonstrates practical implementation of Java programming concepts, Design Patterns, Spring Framework, Spring Boot, Spring Data JPA, Hibernate, Logging, Unit Testing, Mock Testing, Maven, MySQL and Git.

Each exercise is implemented independently with proper project structure and Maven dependency management.

# 🏛 JPA vs Hibernate vs Spring Data JPA

| Feature | JPA | Hibernate | Spring Data JPA |
|---------|-----|-----------|-----------------|
| Type | Specification | ORM Framework | Spring Module |
| Developed By | Oracle/JCP | Red Hat | Spring |
| SQL Generation | ❌ | ✔ | ✔ |
| CRUD Support | Basic | Advanced | Simplified |
| Boilerplate Code | High | Medium | Low |

### Relationship

```
Spring Data JPA
       │
       ▼
      JPA
       │
       ▼
  Hibernate
       │
       ▼
    MySQL
```
