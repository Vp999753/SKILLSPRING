# Spring Framework Learning Exercises

## Overview

This repository contains hands-on implementations of Spring Framework and Spring Boot concepts developed as part of structured learning and practical training. The project focuses on core Spring features such as Dependency Injection, Bean Configuration, Logging, Bean Scopes, and Collection Injection using XML-based configuration.

---

## Technologies Used

- Java
- Spring Framework
- Spring Boot
- Maven
- XML Configuration
- SLF4J Logging

---

## Hands-On Exercises Completed

### Hands-On 1: Spring Boot Project Creation
- Project creation using Spring Initializr
- Maven project setup
- Spring Boot application execution

### Hands-On 2: Loading SimpleDateFormat from Spring Configuration
- Bean definition using XML
- Dependency management through Spring IoC Container

### Hands-On 3: Logging Integration
- Logging configuration using SLF4J
- Application-level logging implementation

### Hands-On 4: Loading Country Bean from XML Configuration
- Bean creation and property injection
- Spring Application Context usage

### Hands-On 5: Singleton and Prototype Scope Demonstration
- Understanding bean lifecycle
- Scope management in Spring
